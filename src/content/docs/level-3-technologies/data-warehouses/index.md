# Data Warehouses

This section covers data warehouse platforms and analytical query engines designed for OLAP workloads.

## Data Warehouse Technologies

- **Snowflake**: Cloud-native, fully-managed data warehouse with elastic scalability and data sharing
- **Databricks**: Unified analytics platform combining data warehousing, engineering, and ML on Apache Spark
- **BigQuery**: Google's serverless, fully-managed data warehouse
- **Redshift**: AWS's columnar data warehouse
- **Impala**: In-memory SQL query engine for Hadoop
- **Greenplum**: Distributed SQL database optimized for analytics

These platforms are designed for analytical queries over large datasets, OLAP operations, and business intelligence use cases. Modern platforms like Snowflake and Databricks offer cloud-native architectures with advanced features like data sharing, lakehouse capabilities, and serverless operations.
