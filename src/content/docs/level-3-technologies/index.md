## Level 3 - Technologies: Tools & Platforms

This section covers the specific technologies and tools used in modern data engineering. It's organized by function area to help you understand the landscape of data engineering tools.

## Technology Categories

- **Batch Processing**: Hadoop, Spark, Hive for large-scale batch data processing
- **Streaming**: Kafka, Flink for real-time data ingestion and processing
- **Cloud**: AWS, GCP, Azure cloud data platforms and services
- **Databases**: BigQuery, Redshift, MongoDB, Cassandra, HBase, DynamoDB, and others
- **Data Formats**: Parquet, Avro, Delta, Hudi, Iceberg for efficient data storage
- **Orchestration**: Airflow, Kubernetes for workflow and infrastructure orchestration
- **Ingestion**: NiFi, Flume, CDC for data ingestion patterns
- **Transformation**: dbt for data transformation and modeling
- **BI Tools**: Tableau, Looker, Superset for visualization and analytics

These technologies form the modern data stack and understanding them is crucial for data engineers.
