## Data Formats: Storage & Serialization

Modern data formats enable efficient storage, compression, and schema evolution.

## Formats Covered

- **Parquet**: Columnar storage format
- **Avro**: Row-based serialization format
- **Delta**: Delta Lake format with ACID semantics
- **Hudi**: Incremental processing format
- **Iceberg**: Table format with versioning
